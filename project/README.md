# Liko 社交移动端网页项目

这是一个使用原生 HTML、CSS、JavaScript 写的类社交 APP 移动端网页项目。项目主要完成了登录、首页帖子展示、发布帖子、个人主页、搜索、消息和私聊等基础功能。

项目没有使用 Vue、React 这类框架，主要是用原生 JS 进行页面切换、事件绑定和接口请求，接口请求部分使用 Axios 进行了简单封装。

ui的设计一开始参考了ins的布局 然后参考了苹果的ui风格 在figma上面参考了一些组件 ui的设计
## 项目功能

### 1. 登录功能

用户可以通过账号和密码登录。登录成功后会保存 token 和 userId，用来访问其他需要登录的页面。

登录页还支持：

- 密码显示 / 隐藏
- 记住账号和密码
- 登录错误提示
- 未登录时自动跳转回登录页
- 如果想退出登录 可以点击个人主页右上角的设置页 那里可以退出登录

### 2. 首页

首页主要分为三个部分：

- For you：推荐帖子
- Follow：关注用户的帖子
- Topic：按话题查看帖子

帖子支持瀑布流展示，可以点击帖子进入详情页，也可以点击头像进入用户主页。

### 3. 帖子详情页

帖子详情页可以查看帖子的完整内容、图片、点赞数和评论。

目前支持：

- 点赞 / 取消点赞
- 收藏 / 取消收藏
- 关注作者
- 查看评论
- 发表评论
- 评论排序
- 删除自己的评论
- 二级评论展示
- 多图轮播

### 4. 发布帖子

用户可以发布自己的帖子，也可以把帖子保存为草稿。

发布时可以填写：

- 标题
- 正文内容
- 图片
- 话题
- 地点
- 可见权限

支持的权限有：

- 草稿
- 公开
- 好友可见
- 仅自己可见

同时也支持编辑已经发布过的帖子和草稿。

### 5. 个人主页

个人主页会展示当前用户的信息，包括头像、背景图、用户名、签名、帖子数、关注数和粉丝数。

用户可以在个人主页中：

- 查看自己发布的帖子
- 查看草稿箱
- 编辑帖子
- 删除帖子
- 进入设置页
- 进入消息页

### 6. 其他用户主页

点击其他用户头像后，可以进入对方主页，查看对方的基本信息和公开帖子。

也可以从对方主页进入私聊页面。

### 7. 搜索功能

搜索页支持输入关键词搜索帖子和用户。

目前支持：

- 搜索按钮
- 清空输入框
- 搜索历史
- 搜索推荐
- 搜索结果展示
- 点击搜索结果跳转到帖子详情或用户主页

### 8. 消息和私聊

消息页会展示不同类型的消息，例如点赞、评论、关注等。

私聊页面使用 WebSocket 实现基础实时聊天功能，可以发送和接收消息，并且会在本地保存最近聊天用户。

### 9. 设置页

设置页支持修改个人资料，包括：

- 头像
- 背景图
- 用户名
- 个性签名
- 退出登录
